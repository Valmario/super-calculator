# Super Calculadora

Criei uma super calculadora capaz de fazer os seguintes cálculos a partir de
dois números:

- Soma;
- Subtração;
- Divisão;
- Multiplicação;
- Potência;
- Raiz quadrada;
- Fatorial;
- Porcentagem;
- Média.

## Objetivo

Exercitar os seguintes conceitos:
1. Implementação de aplicações com HTML, CSS e JavaScript.
2. Manipulação do DOM com JavaScript.
